# IPL Data Analysis (2008 - 2019) 🏏

## Project Overview
This project performs an in-depth **Exploratory Data Analysis (EDA)** on the Indian Premier League (IPL) dataset. By analyzing match results, venue data, and team statistics, the project identifies key factors that influence match outcomes, such as toss decisions and home-ground advantages.

This was developed as a final project for the **Data Visualization** subject to demonstrate proficiency in Python-based data analytics and storytelling through charts.

## Key Insights & Findings
* **Win Margins:** Identified matches won by the maximum runs and wickets, highlighting dominant team performances.
* **Toss Impact:** Analysis of how winning the toss correlates with match victories across different seasons.
* **Venue Excellence:** * **Eden Gardens** and **Wankhede Stadium** hosted the highest number of matches.
    * **Wankhede Stadium** shows a high success rate for defending totals, while other venues favor chasing.
* **Team Performance:** Categorized the "Best Defending Teams" and "Best Chasing Teams" based on historical win data.

## Tech Stack
* **Language:** Python
* **Libraries:**
    * `Pandas`: Data cleaning and CSV handling
    * `Seaborn` & `Matplotlib`: Advanced statistical data visualization
    * `OS`: Directory and file management

## Dataset Information
The analysis uses two primary datasets:
1. `matches.csv`: Contains match-level data (teams, toss, venue, result).
2. `deliveries.csv`: Contains ball-by-ball data for detailed performance metrics.
